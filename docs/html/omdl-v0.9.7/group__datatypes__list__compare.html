Test if all elements of two iterable values are sufficiently equal.

Parameters
v1<iterable> An iterable data type value 1.
v2<iterable> An iterable data type value 2.
p<number> The precision for numerical comparisons.
Returns
(1) <boolean> true when all elements of each iterable value are sufficiently equal and false otherwise.

The iterable values can be of mixed data types. All numerical comparisons are performed using the specified precision. All non-numeric comparisons test for equality. When both v1 and v2 are both numerical vectors, the function almost_eq_nv() provides a more efficient test.

Test to see if two numerical vectors are sufficiently equal.

Parameters
v1<vector-Nd> A vector 1 in an 'N' dimensional space.
v2<vector-Nd> A vector 2 in an 'N' dimensional space.
p<number> The numerical precision.
Returns
(1) <boolean> true when the distance between v1 and v2 is less than d and false otherwise. (2) Returns false when either v1 or v2 are not numerical vectors of the same length.

Can also compare two scalar numbers. To compare general lists of non-numerical values see almost_eq().

Compare the sort order of any two values.

Parameters
v1<value> The values 1.
v2<value> The values 2.
s<boolean> Order ranges by their enumerated sum.
Returns
(1) <integer> An integer value.

Return values (rv) table.

rv order of values
-1 (v2 < v1)
0 (v2 == v1)
+1 (v2 > v1)

The following table summarizes how data type values are ordered.

order type s intra-type ordering
1 undef (singular)
2 number numerical comparison
3 boolean false < true
4 string lexical comparison
5 list compare (1) lengths, then (2) element-wise
6 range true compare sum of range elements
6 range false compare (1) lengths, then (2) element-wise

When comparing two lists of equal length, the comparison continues with successive elements until an ordering can be determined. Two lists are equal when all elements have been compared and no ordering has been determined.

Warning
The performance of element-wise comparisons degrades exponentially with list size and the sum of a range may exceeded the intermediate variable storage capacity for long ranges.
