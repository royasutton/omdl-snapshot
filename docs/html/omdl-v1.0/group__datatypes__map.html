Conventions
  • A map is a list of [key, value] pairs. Keys must be strings.
  • The map variable is always named m; a second map is m2.
  • map_merge(m1, m2): when both maps contain the same key, the value from m1 takes precedence. Order of keys in the result is unspecified.
  • map_update(m, u): keys in u that are also in m update the existing value; keys in u absent from m are appended. Values in m absent from u are preserved unchanged.
  • The special variable $map_strict controls whether map_get_value() asserts on a missing key (strict=true) or returns undef silently (strict=false, the default).
  • map_errors() returns an empty list when the map is valid; a non-empty list indicates the detected error class(es).

Example

Map use script

include <omdl-base.scad>;
map =
[
["part1", ["screw10", [10, 11, 13]]],
["part2", ["screw12", [20, 21, 30]]],
["part3", ["screw10", [10, 10, -12]]],
["config", ["top", "front", "rear"]],
["version", [21, 5, 0]],
["runid", 10]
];
echo( "### map_check ###" );
map_check(map, true);
echo( "### map_exists ###" );
echo( str("is part0 = ", map_exists(map, "part0")) );
echo( str("is part1 = ", map_exists(map, "part1")) );
echo( "### map_get_value ###" );
p1 = map_get_value(map, "part1");
echo( c=second(p1) );
keys = map_get_keys(map);
parts = delete(keys, mv=["config", "version", "runid"]);
echo( "### map_delete ###" );
for ( p = parts )
echo
(
n=p,
p=first(map_get_value(map, p)),
l=second(map_get_value(map, p))
);
echo( "### map_dump ###" );
map_dump(map);
// end_include
