Requires:
include <omdl-base.scad>;

Conventions
  • The primary parameter is always v (the iterable value under test).
  • The comparison value is always cv.
  • All all_* functions return true vacuously for an empty v, except all_lists(), all_strings(), all_numbers(), and all_len(), which require at least one matching element (c > 0) and return false for an empty iterable.
  • The parameter c in all_lists(), all_strings(), all_numbers(), and all_len() is an internal recursion counter. It must not be initialised by callers; doing so produces incorrect results.
  • When v is a non-iterable scalar, functions return the result of applying the predicate directly to v rather than returning undef.
